HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

Elementos Uso Descrição
header done Indica cabeçalho (normalmente com logotipos)
nav done Indica menu de navegação com links para as páginas (só deve haver um por página)
main done Indica conteúdo principal e único, que não se repete
article highlight_off Indica porção do documento que pode ser vista de forma independente
section done Especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|…
aside highlight_off Indica anotações, publicidade ou outros assuntos desconexos
footer done Indica rodapé da secção (com copyright)
details done Define detalhes adicionais que o utilizador pode ver ou esconder, contidos num elemento summary
meter highlight_off Define uma medida escalar estática dentro de um intervalo
progress highlight_off Define uma barra de progresso de uma tarefa
abbr highlight_off Indica o significado de uma abreviatura em conjunto com o atributo title
address highlight_off Informação sobre o contacto do autor

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags "style". Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos

Seletores Uso Descrição
universal highlight_off Aplica o estilo a todos os elementos da página
tipo highlight_off Usa o nome do marcador a estilizar.
classe highlight_off Associa um nome a um estilo
id highlight_off Permite identificar univocamente um elemento do HTML (muito útil no JavaScript)
atributo highlight_off Combina elementos baseado-se no valor de um atributo dado.
pseudo-classe highlight_off É uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado.
pseudo-elemento highlight_off É uma palavra-chave adicionada a um seletor que permite que estilize uma parte específica do elemento selecionado.

Combinadores Uso Descrição
Descendentes highlight_off São separados por espaço
Filhos highlight_off Através do uso de '>'
Adjacentes highlight_off Através do uso de '~'
Imediatamente Adjacentes highlight_off Através do uso de '+'
